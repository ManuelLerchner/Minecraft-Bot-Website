Wood Program:

The second program we are going to look at is a wood-farming program. Lets first look at its helper functions, because we are going to use them in the main program:
The first function is the main gear which drives the bot. As you can see from the code, it evaluates the neighboring blocks at runtime, and checks whether the given block is an oak_log. Then it returns all the blocks found, so that the bot can dynamically decide which block to mine.
