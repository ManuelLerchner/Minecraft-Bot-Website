Task Node:

The Task Node is the most simple node. Its modeled as an abstract class as you can see in the UML-Diagram. Its direct children then implement the actual logic.
When you compile it, it just returns the action asociated with it, and wheter that action can possible fail, thats it. At this level there are no transitions yet because were dealing with a single actions, in other words we are handeling a single states of the state-machine.

An example of a TaskNode is for example a GoToNode which tells the bot to go to the specified coordinates.
