If Node:

Another handy node to have is the IfNode. Similarly to the WhileNode it receives a ConditionNode which, when evaluated, determines which branch of the IfNode the program-flow should follow. Additionally it receives an arbitrary ASTNode for the true-branch and another optional ASTNode for the false-branch.
Like the WhileNode it evaluates the condition of its ConditionNode and then chooses the corresponding branch which should be executed by the bot.
If you call the compile() function on a IfNode it creates the entry- and the exit states of the IfNode and then wires up the two different branches in such a way that the bot can choose between the two paths, according to the ConditionNode.
