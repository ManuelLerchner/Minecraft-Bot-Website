About this Project

In this project I wanted to explore the mineflayer-api which allows an easy creations of minecraft bots.
The project started rather tiedieus because the way youre supposed to program the minecraft bot is by creating a state-machine, aka a deterministic finite automaton.
As you can immagine, creating a statemachine by hand is a very long and and nerve wracking process, even for smaller programms.
For example this is just a small section of a handwritten cobblestone-mine program:

It is obvious that this way of "programming" a minecraft bot isn't very efficient. Yout have have to explicitly tell the program each node and every single transition between those nodes individually.
Its a dull way of programming and the human brain isn't very good at planning such kind of structures.
There has to be a better way.
